# Delaunay Maps GUI (Ap/Ac)

Приложение на Python с графическим интерфейсом для построения двух карт по точкам из Excel:
- карта значений `Ap`
- карта значений `Ac`

Карты строятся методом триангуляции Делоне по координатам `x`, `y`, используют общую шкалу (`белый -> черный`) равными долями и позволяют проверить совпадение изолиний через прозрачное наложение.

## Возможности

- Загрузка файла Excel (`.xlsx`, `.xls`)
- Автоопределение колонок (`rn`, `x`, `y`, `Ap`, `Ac`) + ручной выбор
- Общая триангуляция Делоне для обеих карт
- Отдельные карты `Ap` и `Ac` с colorbar справа и числовыми делениями
- Режим `overlay` с регулировкой прозрачности
- Опция принудительной зеркальности `Ap/Ac` для совпадения изолиний
- Плавные изолинии (интерполяция + сглаживание) с настройкой силы сглаживания
- Переключатели отображения точек и подписей координат
- Подписи названий точек из столбца `rn`
- Встроенная шкала масштаба на картах
- Переключатель отображения изолиний (можно оставить только заливку)
- Инверсия осей `X`/`Y` и режим "поменять X и Y местами"
- Прокрутка левой панели и сворачивание блока параметров
- Экспорт текущего графика в PNG

## Структура проекта

- `app.py` - точка входа
- `ui/main_window.py` - GUI (PySide6)
- `core/data_loader.py` - чтение и подготовка данных Excel
- `core/interpolation.py` - триангуляция и уровни изолиний
- `core/plotting.py` - отрисовка карт и overlay
- `requirements.txt` - зависимости

## Требования

- Python 3.10+
- Установленные зависимости из `requirements.txt`

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

## Использование

1. Нажмите `Загрузить Excel (111.xlsx)` и выберите файл `111.xlsx`.
2. Проверьте соответствие колонок `rn/x/y/Ap/Ac` (или выберите вручную).
3. Нажмите `Построить карты Ap / Ac` для отдельных карт.
4. При необходимости включите:
   - `Плавные изолинии`
   - `Показывать точки`
   - `Показывать координаты точек`
   - `Показывать названия точек (rn)`
   - `Показывать шкалу масштаба`
   - `Инвертировать ось X` / `Инвертировать ось Y`
   - `Поменять X и Y местами`
   и настройте параметр `Сглаживание`.
5. Нажмите `Проверить наложение (overlay)` для проверки совпадения изолиний.
6. Сохраните результат кнопкой `Сохранить текущий график PNG`.

## Формат входных данных

Ожидаются столбцы (имена могут отличаться, если выбрать вручную):
- `rn` - номер точки (опционально)
- `x` - координата X
- `y` - координата Y
- `Ap` - значение поля Ap
- `Ac` - значение поля Ac

Если имена столбцов уже совпадают с ожидаемыми (`rn`, `x`, `y`, `Ap`, `Ac`), поля в интерфейсе подставляются автоматически по умолчанию.

Минимум 3 уникальные точки по `(x, y)` обязателен для триангуляции.
